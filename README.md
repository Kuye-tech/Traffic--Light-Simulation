🚦 Smart Traffic Light Simulation Using Arduino

Overview

This project is a Smart Traffic Light Simulation developed using an Arduino Uno and four LEDs. It demonstrates the basic operation of a traffic light control system by switching LEDs in a predefined sequence with programmed time intervals.

The project was built to strengthen my understanding of Arduino programming, embedded systems, and digital electronics through hands on learning.

---

Features

* Simulates a basic traffic light system.
* Controls four LEDs using Arduino digital output pins.
* Demonstrates timing and sequential logic.
* Simple and beginner friendly design.

---

Components Used

* Arduino Uno
* 4 LEDs
* 220 Ω Resistors
* Breadboard
* Jumper Wires
* USB Cable

---

How It Works

The Arduino executes a program that controls the LEDs according to a timed sequence. Each LED turns on and off to simulate different traffic light states, creating a continuous traffic signal cycle.
